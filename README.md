# Exam Portal (Java Servlet & JSP)

A web-based exam portal developed using Java Servlets and JSP.

## Features
- User Login & Signup
- MCQ-based Exam
- Fullscreen Exam Mode
- Automatic Result Evaluation
- Score Display

## Tech Stack
- Java Servlets
- JSP
- HTML, CSS
- Apache Tomcat
- Eclipse IDE
